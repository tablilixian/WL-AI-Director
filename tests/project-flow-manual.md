## 常见问题排查

### 问题 1: 登录后不显示项目列表

**可能原因**:
- Supabase 连接失败
- 数据库查询出错
- Console 中有错误日志

**排查步骤**:
1. 打开浏览器开发者工具 Console
2. 查看是否有 `[HybridStorage]` 相关错误
3. 检查网络请求是否成功
4. 验证 Supabase 配置是否正确

---

### 问题 2: 数据不同步

**可能原因**:
- 版本号未正确递增
- lastModified 时间戳未更新
- ID 映射表错误

**排查步骤**:
1. 检查 IndexedDB 中的项目数据
2. 检查 Supabase 中的项目数据
3. 对比 version 和 lastModified 字段
4. 查看 Console 中的同步日志

---

### 问题 3: 项目重复显示

**可能原因**:
- 云端数据有重复
- ID 映射表错误
- 去重逻辑失效

**排查步骤**:
1. 检查云端数据是否有重复 ID
2. 清空 ID 映射表 localStorage
3. 刷新页面重新加载
