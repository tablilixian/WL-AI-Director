### 数据模型

```typescript
// 项目
interface Project {
  id: string;
  name: string;
  scriptData: ScriptData;
  shots: Shot[];
  createdAt: Date;
  updatedAt: Date;
}

// 剧本
interface ScriptData {
  content: string;
  visualStyle: string;
  characters: Character[];
  scenes: Scene[];
}

// 镜头
interface Shot {
  id: string;
  actionSummary: string;
  cameraMovement: string;
  keyframes: Keyframe[];
  interval: VideoInterval;
  nineGrid: NineGridData;
}

// 关键帧
interface Keyframe {
  id: string;
  type: 'start' | 'end';
  imageUrl: string;
  visualPrompt: string;
  status: 'pending' | 'generating' | 'completed' | 'failed';
}

// 视频间隔
interface VideoInterval {
  id: string;
  videoUrl: string;
  status: 'pending' | 'generating' | 'completed' | 'failed';
}
```

interface IndexedDBSchema {
  projects: {
    key: string;
    value: Project;
    indexes: { id: string; createdAt: Date };
  };

### 云端存储 (Supabase)

```sql
-- 项目表
CREATE TABLE projects (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id),
  name TEXT NOT NULL,
  script_data JSONB,
  shots JSONB,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- 角色表
CREATE TABLE characters (
  id UUID PRIMARY KEY,
  project_id UUID REFERENCES projects(id),
  name TEXT NOT NULL,
  visual_prompt TEXT,
  turnaround_image TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 场景表
CREATE TABLE scenes (
  id UUID PRIMARY KEY,
  project_id UUID REFERENCES projects(id),
  name TEXT NOT NULL,
  description TEXT,
  reference_image TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
```

## 数据流

### 剧本解析流程

```
用户输入剧本
    ↓
scriptService.parseScript()
    ↓
AI API 调用 (GPT-4/GLM-4)
    ↓
返回解析结果
    ↓
更新项目状态
    ↓
自动保存 (IndexedDB + Supabase)
```

### 图片生成流程

```
用户选择镜头
    ↓
visualService.generateImage()
    ↓
imageAdapter 适配模型
    ↓
AI API 调用 (DALL-E/CogVideoX)
    ↓
返回图片 URL
    ↓
下载并保存到本地 (IndexedDB)
    ↓
更新关键帧状态
    ↓
自动保存
```

### 视频生成流程

```
用户点击生成视频
    ↓
videoService.generateVideo()
    ↓
创建异步任务
    ↓
轮询任务状态
    ↓
任务完成
    ↓
下载视频
    ↓
保存到本地
    ↓
更新镜头状态
    ↓
自动保存
```

## 性能优化

### 1. 自动保存优化
- **Hash 检测**: 避免内容未变化时保存
- **延迟保存**: 1 秒延迟，避免频繁保存
- **批量保存**: 合并多次更新为一次保存

### 2. 图片加载优化
- **本地缓存**: IndexedDB 存储图片
- **Blob URL**: 使用 blob URL 减少网络请求
- **懒加载**: 按需加载图片

### 3. API 调用优化
- **重试机制**: 自动重试失败的请求
- **错误处理**: 详细的错误日志
- **超时控制**: 避免长时间等待

## 安全性

### 1. API Key 管理
- **加密存储**: LocalStorage 加密存储
- **环境变量**: 支持环境变量配置
- **权限控制**: 用户级别的 API Key

### 2. 数据加密
- **传输加密**: HTTPS
- **存储加密**: Supabase 自动加密

### 3. 输入验证
- **XSS 防护**: React 自动转义
- **CSRF 防护**: Supabase 自动处理
- **SQL 注入**: 参数化查询
