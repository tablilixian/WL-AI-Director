## 集成步骤

### 1. Kling 集成步骤

#### 步骤 1: 注册 API Key
- 访问 Kling 官网
- 注册账号
- 获取 API Key

#### 步骤 2: 添加模型配置
在 `types/model.ts` 中添加：

```typescript
{
  id: 'kling',
  name: 'Kling (可灵AI)',
  type: 'video',
  providerId: 'kling',
  apiModel: 'kling-1.5',
  endpoint: '/api/v1/videos/generate',
  description: '可灵AI Kling 视频生成模型，支持运动笔刷，1080P 30帧',
  isBuiltIn: true,
  isEnabled: true,
  params: {
    mode: 'async',
    defaultAspectRatio: '16:9',
    supportedAspectRatios: ['16:9', '9:16', '1:1'],
    defaultDuration: 5,
    supportedDurations: [5, 10],
  },
}
```

#### 步骤 3: 添加提供商配置
在 `types/model.ts` 中添加：

```typescript
{
  id: 'kling',
  name: 'Kling API (kling.kuaishou.com)',
  baseUrl: 'https://api.kling.kuaishou.com',
  isBuiltIn: true,
  isDefault: false,
}
```

#### 步骤 4: 实现 API 调用
在 `services/ai/videoService.ts` 中添加 Kling 的 API 调用逻辑。

### 2. 其他模型集成步骤

参考 Kling 的集成步骤，根据不同模型的 API 文档进行调整。

## 注意事项

### 1. API Key 管理
- 不同模型需要不同的 API Key
- 需要在配置界面添加 API Key 输入框
- 建议加密存储 API Key

### 2. 错误处理
- 需要处理 API 限流
- 需要处理配额不足
- 需要处理网络错误

### 3. 用户体验
- 显示模型加载状态
- 显示生成进度
- 显示错误提示

### 4. 性能优化
- 使用异步生成
- 支持取消生成
- 缓存生成结果
