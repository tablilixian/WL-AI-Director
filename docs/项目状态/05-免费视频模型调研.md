## 集成步骤

### 1. Kling 集成步骤

#### 步骤 1: 注册 API Key
- 访问 Kling 开发者平台: https://app.klingai.com/cn/dev
- 使用手机号或快手扫码登录
- 进入 API Key 管理页面: https://app.klingai.com/cn/dev/api-key
- 创建 API 密钥并命名管理
- 获取 Access Key 和 Secret Key

#### 步骤 2: 生成 JWT Token
- 使用 Access Key 和 Secret Key 生成 JWT Token
- 遵循 JWT (RFC 7519) 标准
- 参考 Kling API 文档进行 JWT 验证

#### 步骤 3: 添加模型配置
在 `types/model.ts` 中添加：

```typescript
{
  id: 'kling',
  name: 'Kling (可灵AI)',
  type: 'video',
  providerId: 'kling',
  apiModel: 'kling-v1-5',
  endpoint: '/v1/videos/generations',
  description: '可灵AI Kling 视频生成模型，支持运动笔刷，1080P 30帧',
  isBuiltIn: true,
  isEnabled: true,
  params: {
    mode: 'async',
    defaultAspectRatio: '16:9',
    supportedAspectRatios: ['16:9', '9:16', '1:1'],
    defaultDuration: 5,
    supportedDurations: [5, 10],
  },
}
```

#### 步骤 4: 添加提供商配置
在 `types/model.ts` 中添加：

```typescript
{
  id: 'kling',
  name: 'Kling API (api-beijing.klingai.com)',
  baseUrl: 'https://api-beijing.klingai.com',
  isBuiltIn: true,
  isDefault: false,
}
```

#### 步骤 5: 实现 API 调用
在 `services/ai/videoService.ts` 中添加 Kling 的 API 调用逻辑：
- 实现 JWT Token 生成
- 添加 Authorization Header: "Bearer {API_TOKEN}"
- 处理异步任务创建和状态查询

### 2. 其他模型集成步骤

参考 Kling 的集成步骤，根据不同模型的 API 文档进行调整。

### 1. API Key 管理
- 不同模型需要不同的 API Key
- 需要在配置界面添加 API Key 输入框
- 建议加密存储 API Key

- 需要处理 API 限流
- 需要处理配额不足
- 需要处理网络错误
