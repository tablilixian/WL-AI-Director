## 环境配置

### 必需工具
- Node.js >= 18

- VS Code（推荐）

### 可选工具
- Supabase CLI（用于本地开发）

## 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/tablilixian/BigBanana-AI-Director.git
cd BigBanana-AI-Director
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量

创建 `.env` 文件：

```env
# Supabase 配置
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

# OpenAI API Key（可选）
VITE_OPENAI_API_KEY=your_openai_api_key

# 智谱 API Key（可选）
VITE_ZHIPU_API_KEY=your_zhipu_api_key

# BigModel API Key（可选）
VITE_BIGMODEL_API_KEY=your_bigmodel_api_key
```

### 4. 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:5173

## 开发规范

### 代码风格

#### TypeScript
- 使用严格模式
- 避免使用 `any` 类型
- 使用接口定义数据结构
- 添加类型注释

#### React
- 使用函数组件
- 使用 Hooks 管理状态
- 避免类组件
- 使用 TypeScript Props 接口

#### 命名规范
- 组件：PascalCase（如 `ShotCard.tsx`）
- 函数：camelCase（如 `generateImage`）
- 常量：UPPER_SNAKE_CASE（如 `API_BASE_URL`）
- 接口：PascalCase（如 `ProjectState`）

### Git 提交规范

#### Commit 消息格式

```
<type>(<scope>): <subject>

<body>

<footer>
```

#### Type 类型
- `feat`: 新功能
- `fix`: 修复 bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 重构代码
- `perf`: 性能优化
- `test`: 测试相关
- `chore`: 构建/工具相关

#### 示例

```bash
git commit -m "feat(shot): 添加镜头拆分功能"
git commit -m "fix(image): 修复本地图片显示问题"
git commit -m "docs(readme): 更新开发指南"
```

## 服务开发

### 创建新服务

#### 1. 创建服务文件

```bash
# 在 services/ 目录下创建新服务
touch services/myService.ts
```

#### 2. 编写服务代码

```typescript
export interface MyServiceConfig {
  apiKey: string;
  baseUrl: string;
}

export class MyService {
  private config: MyServiceConfig;

  constructor(config: MyServiceConfig) {
    this.config = config;
  }

  async fetchData(): Promise<any> {
    const response = await fetch(`${this.config.baseUrl}/data`, {
      headers: {
        'Authorization': `Bearer ${this.config.apiKey}`
      }
    });

    if (!response.ok) {
      throw new Error('请求失败');
    }

    return response.json();
  }
}

export const myService = new MyService({
  apiKey: process.env.VITE_MY_API_KEY || '',
  baseUrl: process.env.VITE_MY_BASE_URL || ''
});
```

#### 3. 使用服务

```typescript
import { myService } from './services/myService';

const fetchData = async () => {
  try {
    const data = await myService.fetchData();
    console.log(data);
  } catch (error) {
    console.error('获取数据失败:', error);
  }
};
```

## 常见问题

### 1. 依赖安装失败

**问题**: `npm install` 失败  
**解决方案**: 清除缓存重试

```bash
rm -rf node_modules package-lock.json
npm install
```

### 2. 端口被占用

**问题**: `npm run dev` 提示端口被占用  
**解决方案**: 更改端口

```bash
# 修改 vite.config.ts
server: {
  port: 3000  // 改为其他端口
}
```

### 3. API Key 无效

**问题**: API 调用失败  
**解决方案**: 检查环境变量

```bash
# 检查 .env 文件
cat .env

# 重新启动开发服务器
npm run dev
```
