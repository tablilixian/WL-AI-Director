## Phase 01: 剧本与分镜

### R-01 智能剧本拆解
- **状态**: ✅ 已完成
- **描述**: 输入小说或故事大纲，AI 自动拆解为包含场次、时间、气氛的标准剧本结构
- **验收**: 输入故事文本 → 输出结构化剧本（场次、角色、场景）

### R-02 视觉化翻译
- **状态**: ✅ 已完成
- **描述**: 自动将文字描述转化为专业的 Midjourney/Stable Diffusion 提示词
- **验收**: 分镜自动生成画面提示词

### R-03 节奏控制
- **状态**: ✅ 已完成
- **描述**: 支持设定目标时长，AI 自动规划镜头密度
- **验收**: 30s/60s/3min 等时长选项

### R-04 手动编辑
- **状态**: ✅ 已完成
- **描述**: 编辑角色视觉描述、分镜画面提示词、角色列表、动作描述和台词
- **验收**: 分镜内容可编辑并保存

---

## Phase 02: 资产与选角

### R-05 角色一致性定妆
- **状态**: ✅ 已完成
- **描述**: 为每个角色生成标准参考图
- **验收**: 角色定妆照生成 + 上传

### R-06 衣橱系统
- **状态**: ✅ 已完成
- **描述**: 支持多套造型（日常、战斗、受伤），基于 Base Look 保持面部特征一致
- **验收**: 角色可添加多个服装变体

### R-07 场景概念设计
- **状态**: ✅ 已完成
- **描述**: 生成环境参考图，确保同一场景下的不同镜头光影统一
- **验收**: 场景概念图生成 + 编辑

### R-08 资产库
- **状态**: ✅ 已完成
- **描述**: 跨项目复用角色/场景/道具资产
- **验收**: 资产加入库、从库替换、跨项目使用

---

## Phase 03: 导演工作台

### R-09 网格化分镜表
- **状态**: ✅ 已完成
- **描述**: 全景式管理所有镜头，按场景分组
- **验收**: 镜头网格视图，状态标签

### R-10 关键帧编辑
- **状态**: ✅ 已完成
- **描述**: 首帧/尾帧生成、上传、预览、复制
- **验收**: 首尾帧管理完整

### R-11 九宫格分镜预览
- **状态**: ✅ 已完成
- **描述**: 一键拆分同一镜头的 9 个视角，先确认描述再生成
- **验收**: 九宫格生成、整图/单格用作首帧

### R-12 上下文感知
- **状态**: ✅ 已完成
- **描述**: AI 生成时自动读取场景图 + 角色服装图
- **验收**: 场景切换、角色管理、服装变体切换

### R-13 视频生成
- **状态**: ✅ 已完成
- **描述**: 单图 Image-to-Video + 首尾帧 Keyframe Interpolation
- **验收**: 支持 Sora/Veo 多模型，横竖屏

---

## Phase 04: 成片与导出

### R-14 实时预览
- **状态**: ✅ 已完成
- **描述**: 时间轴形式预览生成的漫剧片段
- **验收**: 视频播放器，连续播放

### R-15 渲染追踪
- **状态**: ✅ 已完成
- **描述**: 实时监控 API 渲染进度
- **验收**: 渲染日志，进度显示

### R-16 资产导出
- **状态**: ✅ 已完成
- **描述**: 导出关键帧、MP4 片段、EDL、FCPXML、ZIP 项目包
- **验收**: 多种格式导出
