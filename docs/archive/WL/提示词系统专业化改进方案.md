# 提示词系统专业化改进方案

## 📋 概述

本文档详细说明了 WL-AI-Director 提示词系统在漫剧制作专业性方面的改进方案，包括问题分析、具体改进建议、代码示例和实施路线图。

**目标受众**: 开发团队、AI 提示词工程师
**文档版本**: v1.0
**最后更新**: 2026-03-13

---

## 🔍 当前系统评估

### ✅ 优点

1. **覆盖全面**：从剧本解析到视频生成的完整流程
2. **结构清晰**：每个环节都有明确的提示词模板
3. **可扩展性强**：支持多种视觉风格
4. **用户可控**：支持手动编辑和 AI 增强

### ❌ 不足之处（从漫剧制作角度）

| 维度 | 当前状态 | 主要问题 |
|------|---------|---------|
| **风格精度** | 6 种泛化风格 | 缺乏细分风格，过于宽泛 |
| **剧本理解** | 基础结构提取 | 缺乏情绪节拍、视觉节奏 |
| **镜头语言** | 电影镜头术语 | 缺乏漫剧特有镜头语言 |
| **动画质量** | 基础视觉描述 | 缺乏动画原则和关键帧概念 |
| **角色设计** | 外观细节描述 | 缺乏设计语言和剪影识别 |
| **场景设计** | 环境元素描述 | 缺乏环境叙事和空间层次 |

## 1️⃣ 视觉风格提示词 - 增加风格细分

### 📍 当前问题

```typescript
// services/ai/promptConstants.ts
export const VISUAL_STYLE_PROMPTS: { [key: string]: string } = {
  'anime': 'Japanese anime style, cel-shaded, vibrant saturated colors,
  large expressive eyes with detailed iris highlights, dynamic action poses,
  clean sharp outlines, consistent line weight throughout,
  Studio Ghibli/Makoto Shinkai quality, painted sky backgrounds,
  soft ambient lighting with dramatic rim light',
  // ...
};
```

**问题分析**：
- ❌ "Studio Ghibli/Makoto Shinkai quality" 过于宽泛
- ❌ 缺乏具体的视觉参考标准
- ❌ 没有区分不同动漫子风格（少年漫、少女漫、治愈系、战斗系）
- ❌ 缺少可量化的视觉参数

### 🎯 改进方案

// 辅助函数
export const getAnimeSubStyle = (subStyle: string): AnimeSubStyle | null => {
  return ANIME_SUB_STYLES[subStyle] || null;
};

export const getVisualQualityPrompt = (
  lineQuality: string,
  colorStyle: string,
  shadingStyle: string,
  detailLevel: string
): string => {
  const line = VISUAL_QUALITY_METRICS.lineQuality[lineQuality]?.prompt || '';
  const color = VISUAL_QUALITY_METRICS.colorStyle[colorStyle]?.prompt || '';
  const shading = VISUAL_QUALITY_METRICS.shadingStyle[shadingStyle]?.prompt || '';
  const detail = VISUAL_QUALITY_METRICS.detailLevel[detailLevel]?.prompt || '';

  return [line, color, shading, detail].filter(Boolean).join('. ');
};

## 📊 改进优先级

### 🔴 高优先级（立即改进）

1. **增加漫剧风格细分**
   - 少年漫、少女漫、治愈系、赛博朋克等子风格
   - 每种风格的具体视觉参数

2. **增加情绪节拍提取**
   - 剧本解析时提取情绪曲线
   - 识别关键视觉时刻

3. **增加漫剧镜头语言**
   - 速度线、冲击帧、拟声词视觉化
   - 分镜构图指导

### 🟡 中优先级（近期改进）

4. **增加动画原则指导**
   - 挤压拉伸、预备动作、跟随动作
   - 关键帧专业术语

5. **增加角色设计语言**
   - 设计语言、剪影识别度、标志性特征
   - 色彩心理学应用

6. **增加环境叙事**
   - 空间层次、色彩温度、视觉引导
   - 背景故事暗示

### 🟢 低优先级（长期优化）

7. **增加风格参考库**
   - 具体作品风格参考
   - 可视化风格对比

8. **增加质量评估标准**
   - 专业质量检查清单
   - 自动化质量评估

## 🎯 总结

当前提示词系统在**基础功能**方面已经相当完善，但在**漫剧制作专业性**方面还有很大提升空间：

| 维度 | 当前状态 | 改进后 |
|------|---------|---------|
| **风格精度** | 6种泛化风格 | 20+细分风格 + 具体参数 |
| **剧本理解** | 基础结构提取 | 情绪节拍 + 视觉节奏 |
| **镜头语言** | 电影镜头术语 | 漫剧特有镜头语言 |
| **动画质量** | 基础视觉描述 | 动画原则 + 关键帧概念 |
| **角色设计** | 外观细节描述 | 设计语言 + 剪影识别 |
| **场景设计** | 环境元素描述 | 环境叙事 + 空间层次 |

建议优先实施**高优先级**改进，这将显著提升提示词系统在漫剧制作方面的专业性！
