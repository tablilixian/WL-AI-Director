# 图片字段统一修改清单

## 📋 概述

本文档记录了将图片相关字段统一为 `imageUrl` 的所有需要修改的位置。

**修改目标**：
- 统一使用 `imageUrl` 字段存储图片信息
- 删除冗余字段：`referenceImage`, `referenceImageSource`, `localImageId`, `imageUrlSource`
- 支持的 `imageUrl` 格式：
  - `local:img_xxx` - IndexedDB 本地图片
  - `data:image/...` - Base64 编码图片
  - `https://...` - 云端存储图片
  - `video:vid_xxx` - IndexedDB 本地视频
